# Skill: Clawderous

## What
Email-to-anything for OpenClaw. Send commands via email.

## Commands
- /memo - Save note
- /blog - Publish post
- /run - Trigger workflow
- /reply - Send reply
- /status - View stats
- /help - Get help

## Install
```bash
clawhub install clawderous
```

## Config
AGENTMAIL_API_KEY required

## Deploy
Push built files to gh-pages branch
